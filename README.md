# ClinicFlow

Production-ready starter architecture for a clinic workflow SaaS app.

## Stack
- Next.js (App Router) + TypeScript
- Supabase (Auth, Database, Storage, RLS)
- Tailwind CSS
- Zod + React Hook Form
- Server Actions + Route Handlers

## Environment files
- `.env.example`: template for all required keys
- `.env.local`: local development secrets (gitignored)
- `.env.test`: test-only values (when test suite is added)
- `.env.production`: deployment-managed secrets (platform-managed preferred)

## Start point
Create your Next.js app and copy this structure into it, or initialize directly in this folder.
